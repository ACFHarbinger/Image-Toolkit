    subgraph Stage 0: Style Reference & Aesthetic Prior
        IMG_REF(Style Reference) --> CLIP_VIS(CLIP Vision Encode)
        CLIP_VIS --> IP_ADAPT(IP-Adapter)
        AESTHETIC(SDXL Aesthetic Score) -.-> IP_ADAPT
    end

    subgraph Stage 1: Model Stack & Conditioning
        CKPT(Checkpoint: Illustrious XL) --> LORA(LoRA Stack: 3 Slots)
        LORA -- MODEL --> FREEU(FreeU V2)
        LORA -- CLIP --> CLIP_ENC(CLIP Text Encode: Dual Pos/Neg)
    end

    subgraph Stage 2: Base Latent Generation
        LATENT(Empty Latent 1344x768) --> KSAMP_BASE(Base KSampler: dpmpp_2m_sde)
        FREEU --> KSAMP_BASE
        CLIP_ENC --> KSAMP_BASE
        IP_ADAPT -- Conditioning --> KSAMP_BASE
    end

    subgraph Stage 3: Dual-Pass Hires Fix
        KSAMP_BASE -- Latent Tensor --> NNLATENT(NNLatentUpscale x1.5)
        NNLATENT --> KSAMP_HA(Hires Pass A: den. 0.55)
        KSAMP_HA --> KSAMP_HB(Hires Pass B: den. 0.30)
        FREEU -.-> KSAMP_HA
        CLIP_ENC -.-> KSAMP_HA
        FREEU -.-> KSAMP_HB
        CLIP_ENC -.-> KSAMP_HB
    end

    subgraph Stage 4: Multi-Target Detailing
        KSAMP_HB -- Refined Latent --> VAE(VAE Decode Tiled)
        VAE -- Pixel Image --> FD(Face Detailer: YOLOv8)
        FD --> HD(Hand/Body Detailer: YOLOv8)
        FREEU -. Model .-> FD
        CLIP_ENC -. Conditioning .-> FD
        FREEU -. Model .-> HD
        CLIP_ENC -. Conditioning .-> HD
    end

    subgraph Stage 5: Pixel-Space Color Grading
        HD -- Corrected Image --> COLOR(Image Adjustments)
        COLOR --> UNSHARP(Unsharp Mask)
        UNSHARP --> GRAIN(Film Grain)
    end

    subgraph Stage 6: Tiled Super-Resolution & Export
        GRAIN -- Graded Image --> ESRGAN(Tiled ESRGAN: 4x-AnimeSharp)
        ESRGAN -- 8064x4608 Image --> LANCZOS(Lanczos Rescale to Target)
        LANCZOS --> SAVE((Save Wallpaper))
    end
Architectural note on conditioning propagation: The diagram shows a linear flow for clarity, but in the actual graph, Stage 1 outputs fan forward. The LoRA-patched model (post-FreeU) feeds the KSamplers in Stages 2 and 3 and the detailers in Stage 4. The CLIP conditioning feeds all four of those nodes.
